# 版本编号规则
公共版本用于 Git Tag、里程碑和 CURRENT_VERSION。内部微任务使用 `vX.Y.Z-a` 形式，只作为任务检查点，不单独发布或 Tag。所有内部微任务必须可独立编译和回滚。
