# ProjectR Product Pillars

1. 账号会被删除，但关系不会。
2. 玩家是战斗主体，AI 创造机会而不代打。
3. 首脑法令来自本地白名单，必须可理解、可反制、可验证。
4. 长期记忆影响 QTE、支援、对话、画像和现实据点。
5. 实时战斗不等待网络；LLM 有结构化验证和离线回退。

## Scope Gates

- Gate A：无 AI/LLM 时动作战斗成立。
- Gate B：三名 AI 的玩法差异和陪伴感成立。
- Gate C：两轮账号、被针对、删号、被记住的产品闭环成立。
